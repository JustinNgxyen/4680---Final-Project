# 4680---Final-Project
Program Error Explanation Agent.  

---

## How to run:
1. Clone the repo with `git clone https://github.com/JustinNgxyen/4680---Final-Project.git`
2. Add the .env file with `nano .env` in root directory
3. Add to .env `OPENAI_API_KEY=[Insert API key]` and add your API key
4. change into /app with `cd app`
5. activate venv with `source .venv/Scripts/activate`
6. Install packages with `pip install openai python-dotenv`
7. Run app with `python main.py`
