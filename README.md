# 4680---Final-Project
Program Error Explanation Agent.  

---

## How to run:
1. Clone the repo with `git clone https://github.com/JustinNgxyen/4680---Final-Project.git`
2. Open the project in WSL or VSCode and the Bash terminal in VSCode
3. Add the .env file by running `nano .env` in the terminal in the root directory
4. Type in .env `OPENAI_API_KEY=[Insert API key]` and add your API key
5. change into /app with `cd app`
6. activate venv with `source .venv/Scripts/activate`
7. Install packages with `pip install openai python-dotenv`
8. Run app with `python main.py`
