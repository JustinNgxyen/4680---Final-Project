# 4680---Final-Project
Program Error Explanation Agent.  

---

## How to run:
1. Clone the repo with `git clone https://github.com/JustinNgxyen/4680---Final-Project.git`
2. Open the project in VSCode and add a `.env` file in the root directory
3. Type in .env `OPENAI_API_KEY=[Insert API key]` and add your API key
4. change into /app with `cd app`
5. activate venv with `.venv\Scripts\activate` for cmd or `source .venv/Scripts/activate` for bash
6. Install packages with `pip install openai python-dotenv streamlit`
7. Run app with `streamlit run gui.py`
